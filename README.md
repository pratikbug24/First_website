# First Website - E-Learning Platform

Welcome! This is my first website created using HTML, CSS, and JavaScript.

## About

This project is a basic concept for an e-learning course registration platform. It demonstrates fundamental web development skills and provides a foundation for online learning websites.

## Features

- Course catalog with multiple programming languages
- User-friendly navigation
- Responsive design using Bootstrap
- Course enrollment functionality

## Available Courses

The platform offers courses in various programming languages:

- [C Programming](C%20PROGRAMMING.html)
- [HTML Programming](HTML%20PROGRAMMING.html)
- [CSS Programming](css%20programming.html)
- [JavaScript Programming](js%20programming.html)
- [Python](python.html)
- [Java](java.html)
- [PHP Programming](php%20programming.html)

## Getting Started

1. Open [index.html](index.html) in your web browser to view the homepage
2. Navigate through the different course pages
3. Check out the [About](about.html) page to learn more
4. Use the [Contact](contect.html) page for any inquiries

## Project Structure

- `index.html` - Main homepage
- `style.css` - Custom styling
- `about.html` - About page
- `contect.html` - Contact page
- `images/` - Image assets
- `ENROLL/` - Enrollment related pages
- Various course-specific HTML pages

## Technologies Used

- **HTML5** - Structure and content
- **CSS3** - Styling and layout
- **JavaScript** - Interactive functionality
- **Bootstrap 5.3.3** - Responsive framework
